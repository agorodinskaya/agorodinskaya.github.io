Final project and next steps:

For my final project I created Facemood. It recognises the user's face and plays music based on the chosen mood, projecting it on Raspberry pi. The colour of the background can be changed using a voice interface. I got the idea of using Raspberry pi from one of the Meetups I've been to a few weeks ago. I enjoyed working on this project and am planning to add more features to it, especially using Raspberry pi. For example I would like to add lights based on the user's mood.
